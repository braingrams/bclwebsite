Release Year 2018

Mayday Safety was developed under expert guidance of the US Special Operations veterans and security, medical and tactical professionals.

The Description

With the number of school shootings in the US reaching menacing proportions, an urgent need for a simplified incident response emerged. Mayday Safety came up with an idea to convert a battlefield tested perimeter notification and communication system into civilian use for schools and business organizations.

The Challenge

Mayday Safety approached Mercury Development to build a solution that would include mobile apps for personal usage, a web admin dashboard to manage organizations and a separate web tool for superadmins. Besides, Mayday Safety also aimed to provide a quick and easy way to reach out to families when all clear.

The Solution

Mercury Development BA team started with eliciting and detailing a complete and comprehensive list of features for the Development team to build all the components required for the solution to work and save people’s lives: a native mobile app for iOS, admin and super admin web interfaces and a scalable backend.

Distress signals

Allows sending distress signals to a preconfigured group of contacts or to all users nearby depending on the system settings.

Safety signals

Supports sending safety signals to instantly inform the family members when the user is beyond danger.

Two-factor authentication

Enjoys highly secure user authentication: touch ID is required and verification codes are sent via SMS.

Web Admin CMS

The Admin portal was built for system management, including setting boundaries of the perimeter, selecting the notifications type, and others, as well as for real time user location tracking on a multi-colored map.

Easy to use

Great Features

Precise geopositioning to track distress signals

All user types are covered by the tracking system

24/7 live monitoring in full-screen mode

Reimbursed by the company, provided to end users for free

The Result

Mayday Safety™, the world’s top personal safety app, helps empower over 22,000 people in 85 countries.
